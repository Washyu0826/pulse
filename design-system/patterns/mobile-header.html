行動版 Header 提案

現況在 ≤375px 溢出（logo 96px＋三連結＋Live ≈ 376px）→ 整頁橫向捲動、sticky 錯位。兩案：A 精簡頂列（最小改動）、B 底部 tab bar（正解，拇指可達性最好）。「Live」恆亮綠點改成誠實的「更新於 HH:MM」。

藏 wordmark 標語與 Live、導覽縮短字、全部 44px 命中

規格

  • tab 高 56px ＋ pb: env(safe-area-inset-bottom)，icon 21px ＋ 11px 標籤。
  • 現用 tab：#385CCC ＋ 600 字重——不只變色（aria-current="page"）。
  • 「更新於 08:30」由 API 的 last_crawled_at 帶出，取代恆亮 Live 的過度承諾。
  • 收藏 tab 可加未打包計數 badge（打包 nudge 的入口）。
  • sticky 頂列改純半透明底（低階 Android blur 合成有感時降級）。
